## **🚀 HOW TO RUN ENTERPRISE DASHBOARD**

### **Step 1: Install Enterprise Dependencies**
```bash
pip install streamlit plotly pandas numpy
```

### **Step 2: Run Enterprise Dashboard**
```bash
cd trade-dashboard
streamlit run enterprise_dashboard.py
```

### **Step 3: Access at**
- **Local**: http://localhost:8501
- **Network**: http://192.168.5.88:8501
